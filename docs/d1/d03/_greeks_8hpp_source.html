24/**
25 * Greeks event is a snapshot of the option price, Black-Scholes volatility and greeks.
26 * It represents the most recent information that is available about the corresponding values on
27 * the market at any given moment of time.
28 *
29 * <h3><a name="eventFlagsSection">Event flags, transactions and snapshots</a></h3>
30 *
31 * Some greeks sources provide a consistent view of the set of known greeks.
32 * The corresponding information is carried in @ref ::getEventFlags() "eventFlags" property.
33 * The logic behind this property is detailed in IndexedEvent class documentation.
34 * Multiple event sources for the same symbol are not supported for greeks, thus
35 * @ref ::getSource() "source" property is always @ref IndexedEventSource::DEFAULT "DEFAULT".
36 *
37 * TimeSeriesEventModel class handles all the snapshot and transaction logic and conveniently represents
38 * a list current of time-series events order by their @ref ::getTime() "time".
39 * It relies on the code of AbstractIndexedEventModel to handle this logic.
40 * Use the source code of AbstractIndexedEventModel for clarification on transactions and snapshot logic.
41 *
42 * <h3>Publishing Greeks</h3>
43 *
44 * Publishing of greeks events follows the general rules explained in TimeSeriesEvent class documentation.
45 */
