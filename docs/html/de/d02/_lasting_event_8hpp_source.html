12/**
13 * Represents up-to-date information about some condition or state of an external entity that updates in real-time.
14 * For example, a Quote is an up-to-date information about best bid and best offer for a specific symbol.
15 *
16 * Lasting events are conflated for each symbol. Last event for each symbol is always delivered to event listeners on
17 * subscription, but intermediate (next-to-last) events are not queued anywhere, they are simply discarded as stale
18 * events. More recent events represent an up-to-date information about some external entity.
19 *
20 * Lasting events can be used with DXFeed::getLastEvent() and DXFeed::getLastEvents methods to retrieve last events
21 * for each symbol.
22 *
23 * Note, that subscription to all lasting events of a specific type via WildcardSymbol::ALL symbol object does not
24 * benefit from the above advantages of lasting events.
25 */
26struct DXFCPP_EXPORT LastingEvent {
27 /// The alias to a type of shared pointer to the LastingEvent object
28 using Ptr = std::shared_ptr<LastingEvent>;
29};
