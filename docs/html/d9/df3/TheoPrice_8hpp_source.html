26 /**
27  * Theo price is a snapshot of the theoretical option price computation that is
28  * periodically performed by <a href="http://www.devexperts.com/en/products/price.html">dxPrice</a>
29  * model-free computation.
30  * It represents the most recent information that is available about the corresponding
31  * values at any given moment of time.
32  * The values include first and second order derivative of the price curve by price, so that
33  * the real-time theoretical option price can be estimated on real-time changes of the underlying
34  * price in the vicinity.
35  *
36  * <h3><a name="eventFlagsSection">Event flags, transactions and snapshots</a></h3>
37  *
38  * Some TheoPrice sources provide a consistent view of the set of known TheoPrice.
39  * The corresponding information is carried in @ref ::getEventFlags() "eventFlags" property.
40  * The logic behind this property is detailed in IndexedEvent class documentation.
41  * Multiple event sources for the same symbol are not supported for TheoPrice, thus
42  * @ref ::getSource() "source" property is always @ref IndexedEventSource::DEFAULT "DEFAULT".
43  *
44  * <p>TimeSeriesEventModel class handles all the snapshot and transaction logic and conveniently represents
45  * a list current of time-series events order by their @ref ::getTime() "time".
46  *
47  * <h3>Publishing TheoPrice</h3>
48  *
49  * Publishing of TheoPrice events follows the general rules explained in TimeSeriesEvent class
50  * documentation.
51  *
52  * <h3>Implementation details</h3>
53  *
54  * This event is implemented on top of QDS records `TheoPrice`.
55  */
