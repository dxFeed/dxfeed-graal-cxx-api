24 /**
25  * Theo price is a snapshot of the theoretical option price computation that is
26  * periodically performed by <a href="http://www.devexperts.com/en/products/price.html">dxPrice</a>
27  * model-free computation.
28  * It represents the most recent information that is available about the corresponding
29  * values at any given moment of time.
30  * The values include first and second order derivative of the price curve by price, so that
31  * the real-time theoretical option price can be estimated on real-time changes of the underlying
32  * price in the vicinity.
33  *
34  * <h3><a name="eventFlagsSection">Event flags, transactions and snapshots</a></h3>
35  *
36  * Some TheoPrice sources provide a consistent view of the set of known TheoPrice.
37  * The corresponding information is carried in @ref ::getEventFlags() "eventFlags" property.
38  * The logic behind this property is detailed in IndexedEvent class documentation.
39  * Multiple event sources for the same symbol are not supported for TheoPrice, thus
40  * @ref ::getSource() "source" property is always @ref IndexedEventSource::DEFAULT "DEFAULT".
41  *
42  * <p>TimeSeriesEventModel class handles all the snapshot and transaction logic and conveniently represents
43  * a list current of time-series events order by their @ref ::getTime() "time".
44  *
45  * <h3>Publishing TheoPrice</h3>
46  *
47  * Publishing of TheoPrice events follows the general rules explained in TimeSeriesEvent class
48  * documentation.
49  *
50  * <h3>Implementation details</h3>
51  *
52  * This event is implemented on top of QDS records `TheoPrice`.
53  */
