\doxysubsubsection{Detailed Description}
Price type attribute of \mbox{\hyperlink{structCandleSymbol}{Candle\+Symbol}} defines price that is used to build the candles. 

\doxysubparagraph*{Implementation details}

This attribute is encoded in a symbol string with \mbox{\hyperlink{structMarketEventSymbols_a87a94614e292548132480d85a1dfbe9d}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}}, \mbox{\hyperlink{structMarketEventSymbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}}, and \mbox{\hyperlink{structMarketEventSymbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key}} methods. The key to use with these methods is available via \mbox{\hyperlink{structCandlePrice_ae38797977ef63681da68c45ffeaa1450}{Candle\+Price\+::\+ATTRIBUTE\+\_\+\+KEY}} constant. The value that this key shall be set to is equal to the corresponding \mbox{\hyperlink{structCandlePrice_ae93d8c3b9b9d4a45f940797bbf7cdfcb}{Candle\+Price\+::to\+String()}}
