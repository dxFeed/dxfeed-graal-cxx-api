Address may be just "{}$<$host$>$\+:$<$port$>$"{} of server, URL, or a file path. The "{}\mbox{[}update=$<$period$>$\mbox{]}"{} clause can be optionally added at the end of the address to specify an \doxylink{class_instrument_profile_connection_a3828118661a663677a0e104ef1c32732}{update period} via an address string.
