\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_candle}{Candle} alignment attribute of \doxylink{structdxfcpp_1_1_candle_symbol}{Candle\+Symbol} defines how candle are aligned with respect to time. 

\doxyparagraph*{Implementation details}

The key to use with these methods is available via \doxylink{structdxfcpp_1_1_candle_alignment_a9c3c16aad08be954175dc6b05a0f80f8}{Candle\+Alignment\+::\+ATTRIBUTE\+\_\+\+KEY} constant.
