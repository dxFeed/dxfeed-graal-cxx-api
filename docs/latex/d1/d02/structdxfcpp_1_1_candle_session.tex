\doxysubsubsection{Detailed Description}
\doxylink{structdxfcpp_1_1_session}{Session} attribute of \doxylink{structdxfcpp_1_1_candle_symbol}{Candle\+Symbol} defines trading that is used to build the candles. 

\doxyparagraph*{Implementation details}

This attribute is encoded in a symbol string with \doxylink{structdxfcpp_1_1_market_event_symbols_a20d43c86ae4261b48eae338b9a53e3cf}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}, \doxylink{structdxfcpp_1_1_market_event_symbols_a05f9118ae0d921489495dec94159e8f0}{change\+Attribute\+String\+By\+Key}, and \doxylink{structdxfcpp_1_1_market_event_symbols_a0f662cf613e3e6decd079d9782d9742d}{remove\+Attribute\+String\+By\+Key} methods.

\doxylink{structdxfcpp_1_1_candle_session_ab7e6e4a28535cf8b18fb811be3687031}{Candle\+Session\+::\+ANY} session is a default. The key to use with these methods is available via \doxylink{structdxfcpp_1_1_candle_session_adfd0e3e15583968f4af20d2cb1ed9c75}{Candle\+Session\+::\+ATTRIBUTE\+\_\+\+KEY} constant. The value that this key shall be set to is equal to the corresponding \doxylink{structdxfcpp_1_1_candle_session_a72c6c934b4d3f1598b772429f8458fc6}{Candle\+Session\+::to\+String()}
