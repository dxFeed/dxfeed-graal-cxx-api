\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_series}{Series} event is a snapshot of computed values that are available for all option series for a given underlying symbol based on the option prices on the market. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxylink{classdxfcpp_1_1_series}{Series} is an \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} with multiple instances of event available for each underlying symbol. Each series event instance corresponds to an Option\+Series of the corresponding underlying. The correspondence between a series event instance and an Option\+Series is established via \doxylink{classdxfcpp_1_1_series_a3bc261f83f17749d4ae046b24a6120fd}{expiration} property. If case where there are multiple series at the same expiration day id, then series events are are ordered by their \doxylink{classdxfcpp_1_1_series_ac6144eb907c3f037c8eda1d8b08de5d0}{index} in the same order as the corresponding Option\+Series are ordered by their attributes.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

\doxylink{classdxfcpp_1_1_series}{Series} data source provides a consistent view of the set of known series. The corresponding information is carried in \doxylink{classdxfcpp_1_1_series_aa9c1ea427a9724e12fbfbb39315fbec5}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for series, thus \doxylink{classdxfcpp_1_1_series_aaaa8e2016b3aff845df96c386f24f396}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

Indexed\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of events.

\doxyparagraph*{Implementation details}

This event is implemented on top of QDS records {\ttfamily \doxylink{classdxfcpp_1_1_series}{Series}}.
