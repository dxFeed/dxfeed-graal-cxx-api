\doxysubsubsection{Detailed Description}
Price type attribute of \doxylink{struct_candle_symbol}{Candle\+Symbol} defines price that is used to build the candles. 

\doxysubsubsubsubsection*{Implementation details}

This attribute is encoded in a symbol string with \doxylink{struct_market_event_symbols_a8eea112429ea61730be80e4fac021095}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}, \doxylink{struct_market_event_symbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}, and \doxylink{struct_market_event_symbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key} methods. The key to use with these methods is available via \doxylink{struct_candle_price_ae38797977ef63681da68c45ffeaa1450}{Candle\+Price\+::\+ATTRIBUTE\+\_\+\+KEY} constant. The value that this key shall be set to is equal to the corresponding \doxylink{struct_candle_price_aa48e720eb3df20b9b1f50e03927fc945}{Candle\+Price\+::to\+String()}
