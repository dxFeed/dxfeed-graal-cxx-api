\doxysubsubsection{Detailed Description}
Option Sale event represents a trade or another market event with the price (for example, market open/close price, etc.) for each option symbol listed under the specified \mbox{\hyperlink{classUnderlying}{Underlying}}. 

Option Sales are intended to provide information about option trades {\bfseries{in a continuous time slice}} with the additional metrics, like Option Volatility, Option Delta, and \mbox{\hyperlink{classUnderlying}{Underlying}} Price.

Option Sale events have unique \mbox{\hyperlink{classOptionSale_a944fc7602702af9149559f29d195cedd}{index}} which can be used for later correction/cancellation processing.

Option sale data source provides a consistent view of the set of known option sales. The corresponding information is carried in \mbox{\hyperlink{classOptionSale_ad3795401d748d7151cde6fe71ef168cc}{event\+Flags}} property. The logic behind this property is detailed in \mbox{\hyperlink{structIndexedEvent}{Indexed\+Event}} class documentation. Multiple event sources for the same symbol are not supported for option sale events, thus \mbox{\hyperlink{classOptionSale_a1d2305856f07f2718579d75062765eff}{source}} property is always \mbox{\hyperlink{classIndexedEventSource_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}}.

This event is implemented on top of QDS records {\ttfamily \mbox{\hyperlink{classOptionSale}{Option\+Sale}}}.
