\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{structSession}{Session}} attribute of \mbox{\hyperlink{structCandleSymbol}{Candle\+Symbol}} defines trading that is used to build the candles. 

\doxysubparagraph*{Implementation details}

This attribute is encoded in a symbol string with \mbox{\hyperlink{structMarketEventSymbols_a87a94614e292548132480d85a1dfbe9d}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}}, \mbox{\hyperlink{structMarketEventSymbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}}, and \mbox{\hyperlink{structMarketEventSymbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key}} methods.

\mbox{\hyperlink{structCandleSession_a67d891c75a8f8571d1c63d09615c98c6}{Candle\+Session\+::\+ANY}} session is a default. The key to use with these methods is available via \mbox{\hyperlink{structCandleSession_a9711ee826ebf3527e69f5e01725ca5e4}{Candle\+Session\+::\+ATTRIBUTE\+\_\+\+KEY}} constant. The value that this key shall be set to is equal to the corresponding \mbox{\hyperlink{structCandleSession_a179a6e0bc362ec0965acb621ec504288}{Candle\+Session\+::to\+String()}}
