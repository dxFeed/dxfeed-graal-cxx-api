\doxysubsubsection{Detailed Description}
Option Sale event represents a trade or another market event with the price (for example, market open/close price, etc.) for each option symbol listed under the specified \doxylink{class_underlying}{Underlying}. 

Option Sales are intended to provide information about option trades {\bfseries{in a continuous time slice}} with the additional metrics, like Option Volatility, Option Delta, and \doxylink{class_underlying}{Underlying} Price.

Option Sale events have unique \doxylink{class_option_sale_a944fc7602702af9149559f29d195cedd}{index} which can be used for later correction/cancellation processing.

Option sale data source provides a consistent view of the set of known option sales. The corresponding information is carried in \doxylink{class_option_sale_ad3795401d748d7151cde6fe71ef168cc}{event\+Flags} property. The logic behind this property is detailed in \doxylink{struct_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for option sale events, thus \doxylink{class_option_sale_a2a581fb5d0320601bd7fe7736b138d2f}{source} property is always \doxylink{class_indexed_event_source_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}.

This event is implemented on top of QDS records {\ttfamily \doxylink{class_option_sale}{Option\+Sale}}.
