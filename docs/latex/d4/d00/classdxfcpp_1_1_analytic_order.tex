\doxysubsubsection{Detailed Description}
Represents an extension of \doxylink{classdxfcpp_1_1_order}{Order} introducing analytic information, e.\+g. 

adding to this order iceberg related information (\doxylink{classdxfcpp_1_1_analytic_order_addb54c9116da813afb4977be1a397edb}{iceberg\+Peak\+Size}, \doxylink{classdxfcpp_1_1_analytic_order_a4cc58d9fab2f2d133d4a6ede581ff9be}{iceberg\+Hidden\+Size}, \doxylink{classdxfcpp_1_1_analytic_order_a5c61a823f7b121e25d92613c7726b9f2}{iceberg\+Executed\+Size}). The collection of analytic order events of a symbol represents the most recent analytic information that is available about orders on the market at any given moment of time.

Analytic order is similar to a regular \doxylink{classdxfcpp_1_1_order}{Order}. In addition this event has few additional properties\+: 
\begin{DoxyItemize}
\item \doxylink{classdxfcpp_1_1_analytic_order_addb54c9116da813afb4977be1a397edb}{iceberg\+Peak\+Size} -\/ the size of the peak, i.\+e. the visible part of the iceberg, that is being continually refilled until the order is fully traded or cancelled; 
\item \doxylink{classdxfcpp_1_1_analytic_order_a4cc58d9fab2f2d133d4a6ede581ff9be}{iceberg\+Hidden\+Size} -\/ the prediction of current hidden size of the iceberg, as inferred by the model; 
\item \doxylink{classdxfcpp_1_1_analytic_order_a5c61a823f7b121e25d92613c7726b9f2}{iceberg\+Executed\+Size} -\/ the executed size of the iceberg order. For \doxylink{structdxfcpp_1_1_iceberg_type_a0182dd8f548bff25aba7b9b598b94ae4}{Iceberg\+Type\+::\+SYNTHETIC} type represents total executed size of all orders composing current iceberg; 
\item \doxylink{classdxfcpp_1_1_analytic_order_a862b8dc95362052bde3464b02fd86384}{Analytic\+Order\+::get\+Iceberg\+Type()} -\/ type of the iceberg, either native (exchange-\/managed) or synthetic (managed outside of the exchange). 
\end{DoxyItemize}

\doxyparagraph*{Implementation details}

This event is implemented on top of QDS records {\ttfamily \doxylink{classdxfcpp_1_1_analytic_order}{Analytic\+Order}\#\texorpdfstring{$<$}{<}source-\/id\texorpdfstring{$>$}{>}}, where {\ttfamily \texorpdfstring{$<$}{<}source-\/id\texorpdfstring{$>$}{>}} is up to 4 ASCII characters with a mnemonic for the source like "{}\+GLBX"{}.
