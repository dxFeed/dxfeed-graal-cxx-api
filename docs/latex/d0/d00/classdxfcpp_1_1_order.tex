\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_order}{Order} event is a snapshot for a full available market depth for a symbol. 

The collection of order events of a symbol represents the most recent information that is available about orders on the market at any given moment of time. \doxylink{classdxfcpp_1_1_order}{Order} events give information on several levels of details, called scopes -\/ see \doxylink{structdxfcpp_1_1_scope}{Scope}. \doxylink{structdxfcpp_1_1_scope}{Scope} of an order is available via \doxylink{classdxfcpp_1_1_order_base_a9504b4283cea506a30b47c148318d31d}{scope} property.

\doxylink{classdxfcpp_1_1_order}{Order} events arrive from multiple sources for the same market symbol and are distinguished by their \doxylink{structdxfcpp_1_1_indexed_event_ad5a71500687f8cebb2a15964144638dc}{index}. Index is a unique per symbol identifier of the event. It is unique across all the sources of depth information for the symbol. The event with \doxylink{classdxfcpp_1_1_order_base_a3772a63a9d2a79b8cb9a687a2b2c0a7a}{size} either {\ttfamily 0} or {\ttfamily NaN} is a signal to remove previously received order for the corresponding index. The method \doxylink{classdxfcpp_1_1_order_base_aa7c935513d5717625af3563761703350}{Order\+::has\+Size()} is a convenient method to test for size presence.

Events from finer-\/grained \doxylink{structdxfcpp_1_1_scope}{Scope} of detail give more information and include events from coarse-\/grained \doxylink{structdxfcpp_1_1_scope}{Scope} of detail. For a consistent representation of the market depth to the end-\/user, order events must be filtered to remove coarse-\/grained events that are superseded by the finer-\/grained events.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some order event sources provide a consistent view of the price-\/level or detailed order book. Their updates may incorporate multiple changes to price levels or to individual orders that have to be processed at the same time. The corresponding information is carried in \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{event\+Flags} property.

See {\ttfamily Event Flags} section of \doxylink{classdxfcpp_1_1_order_base}{Order\+Base} class documentation for details.

The composite quotes with \doxylink{structdxfcpp_1_1_scope_a9fc4da1827b15f2ce0214a2b8474e324}{Scope\+::\+COMPOSITE} and regional quotes with \doxylink{structdxfcpp_1_1_scope_a4f46e8856351acacfa126a2b37c1ab2e}{Scope\+::\+REGIONAL} come individually from different venues and are not related to each other in any transactional way. The result of \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{Order\+::get\+Event\+Flags} method for them is always zero.

\doxyparagraph*{Publishing order books}

When publishing an order event with \doxylink{structdxfcpp_1_1_d_x_publisher_a9ec3ba1e21e011c21c5d9f30c21305f1}{DXPublisher\+::publish\+Events()} method, least significant 32 bits of order \doxylink{structdxfcpp_1_1_indexed_event_ad5a71500687f8cebb2a15964144638dc}{index} must be in a range of from 0 to {\ttfamily std\+::numeric\+\_\+limits\texorpdfstring{$<$}{<}std\+::int32\+\_\+t\texorpdfstring{$>$}{>}::\doxylink{structdxfcpp_1_1_shared_entity_aa117214d0b4e4fc6979400eff315a8d3}{max()}} inclusive. Use \doxylink{classdxfcpp_1_1_order_base_a1813c11582e335c3654c028ad80b7a0d}{Order\+::set\+Source()} method after \doxylink{structdxfcpp_1_1_indexed_event_a5467b31a73a69502b6590656980b108d}{Order\+::set\+Index()} to properly include source identifier into the index.

A snapshot has to be published in the {\itshape descending} order of \doxylink{structdxfcpp_1_1_indexed_event_ad5a71500687f8cebb2a15964144638dc}{index}, starting with an event with the largest index and marking it with \doxylink{classdxfcpp_1_1_event_flag_a440532930f74605794158bb59a161c59}{Event\+Flag\+::\+SNAPSHOT\+\_\+\+BEGIN} bit in \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{event\+Flags}, and finishing the snapshot with an event that has zero 32 least significant bits of index. \doxylink{classdxfcpp_1_1_event_flag_acd462d106e5f20a5fe315bca6cdc31e0}{Event\+Flag\+::\+SNAPSHOT\+\_\+\+END} bit in \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{event\+Flags} is optional during publishing. It will be properly set on receiving end anyway.

\doxyparagraph*{Limitations}

This event type cannot be used with \doxylink{structdxfcpp_1_1_shared_entity_aa117214d0b4e4fc6979400eff315a8d3}{DXFeed\+::get\+Last\+Event()} method.
