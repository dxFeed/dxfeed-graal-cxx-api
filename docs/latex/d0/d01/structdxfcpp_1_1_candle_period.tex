\doxysubsubsection{Detailed Description}
Period attribute of \doxylink{structdxfcpp_1_1_candle_symbol}{Candle\+Symbol} defines aggregation period of the candles. 

Aggregation period is defined as pair of a \doxylink{structdxfcpp_1_1_candle_period_a81e8e7abc6a7ebd418559ff00cb2d515}{value} and \doxylink{structdxfcpp_1_1_candle_period_a3aa05f7949e7f5337f6c9b02ccb57bd5}{type}.

\doxyparagraph*{Implementation details}

This attribute is encoded in a symbol string with \doxylink{structdxfcpp_1_1_market_event_symbols_a20d43c86ae4261b48eae338b9a53e3cf}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}, \doxylink{structdxfcpp_1_1_market_event_symbols_a05f9118ae0d921489495dec94159e8f0}{change\+Attribute\+String\+By\+Key}, and \doxylink{structdxfcpp_1_1_market_event_symbols_a0f662cf613e3e6decd079d9782d9742d}{remove\+Attribute\+String\+By\+Key} methods. The key to use with these methods is available via \doxylink{structdxfcpp_1_1_candle_period_a0719bbfc3999fdead85dcbbc8dc17df0}{Candle\+Period\+::\+ATTRIBUTE\+\_\+\+KEY} constant. The value that this key shall be set to is equal to the corresponding \doxylink{structdxfcpp_1_1_candle_period_aa56a1de2f20df3c94a5920a046bb285f}{Candle\+Period\+::to\+String()} "{}\+Candle\+Period.\+to\+String()"{}
