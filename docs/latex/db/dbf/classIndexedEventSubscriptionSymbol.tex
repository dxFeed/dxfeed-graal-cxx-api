\doxysubsubsection{Detailed Description}
Represents subscription to a specific source of indexed events. 

This is symbol is observed by Observable\+Subscription\+Change\+Listener methods symbols\+Added and symbols\+Removed when subscription to \mbox{\hyperlink{structIndexedEvent}{Indexed\+Event}} is defined.

Instances of this class can be used with \mbox{\hyperlink{classDXFeedSubscription}{DXFeed\+Subscription}} to specify subscription to a particular source of indexed events. By default, when subscribing to indexed events by their event symbol object, the subscription is performed to all supported sources.

\doxysubparagraph*{Equality and hash codes}

Indexed event subscription symbols are compared based on their \mbox{\hyperlink{classIndexedEventSubscriptionSymbol_a261f514071ad68164ff238f9bb34f775}{event\+Symbol}} and \mbox{\hyperlink{classIndexedEventSubscriptionSymbol_a61dfc08b2b581362c71daf7d56e2c71e}{source}}.
