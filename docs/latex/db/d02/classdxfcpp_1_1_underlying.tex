\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_underlying}{Underlying} event is a snapshot of computed values that are available for an option underlying symbol based on the option prices on the market. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \doxylink{classdxfcpp_1_1_underlying}{Underlying} sources provide a consistent view of the set of known \doxylink{classdxfcpp_1_1_underlying}{Underlying} events. The corresponding information is carried in \doxylink{classdxfcpp_1_1_underlying_a33ac7e578de3e04991b67bddd3bacd25}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for \doxylink{classdxfcpp_1_1_underlying}{Underlying}, thus \doxylink{classdxfcpp_1_1_underlying_ad854e8cc9d69f40573e1ebc8da09b54b}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{classdxfcpp_1_1_underlying_a976f14e478a007a93eb6b0b4a5e71aab}{time}.

\doxyparagraph*{Publishing \doxylink{classdxfcpp_1_1_underlying}{Underlying}}

Publishing of \doxylink{classdxfcpp_1_1_underlying}{Underlying} events follows the general rules explained in \doxylink{structdxfcpp_1_1_time_series_event}{Time\+Series\+Event} class documentation.

\doxyparagraph*{Implementation details}

This event is implemented on top of QDS record {\ttfamily \doxylink{classdxfcpp_1_1_underlying}{Underlying}}.
