\doxysubsubsection{Detailed Description}
Theo price is a snapshot of the theoretical option price computation that is periodically performed by \href{http://www.devexperts.com/en/products/price.html}{\texttt{ dx\+Price}} model-\/free computation. 

It represents the most recent information that is available about the corresponding values at any given moment of time. The values include first and second order derivative of the price curve by price, so that the real-\/time theoretical option price can be estimated on real-\/time changes of the underlying price in the vicinity.

\doxysubparagraph*{\label{_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \mbox{\hyperlink{classTheoPrice}{Theo\+Price}} sources provide a consistent view of the set of known \mbox{\hyperlink{classTheoPrice}{Theo\+Price}}. The corresponding information is carried in event\+Flags property. The logic behind this property is detailed in \mbox{\hyperlink{structIndexedEvent}{Indexed\+Event}} class documentation. Multiple event sources for the same symbol are not supported for \mbox{\hyperlink{classTheoPrice}{Theo\+Price}}, thus source property is always \mbox{\hyperlink{classIndexedEventSource_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their time.

\doxysubparagraph*{Publishing \mbox{\hyperlink{classTheoPrice}{Theo\+Price}}}

Publishing of \mbox{\hyperlink{classTheoPrice}{Theo\+Price}} events follows the general rules explained in \mbox{\hyperlink{structTimeSeriesEvent}{Time\+Series\+Event}} class documentation.

\doxysubparagraph*{Implementation details}

This event is implemented on top of QDS records {\ttfamily \mbox{\hyperlink{classTheoPrice}{Theo\+Price}}}.
