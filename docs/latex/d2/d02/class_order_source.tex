\doxysubsubsection{Detailed Description}
Identifies source of \doxylink{class_order}{Order}, \doxylink{class_analytic_order}{Analytic\+Order}, \doxylink{class_otc_markets_order}{Otc\+Markets\+Order} and \doxylink{class_spread_order}{Spread\+Order} events. 

There are the following kinds of order sources\+: 
\begin{DoxyItemize}
\item {\itshape Synthetic} sources \doxylink{class_order_source_af9c7de7588af5d798c2857af264c4e47}{Order\+Source\+::\+COMPOSITE\+\_\+\+BID}, \doxylink{class_order_source_a7197bc1f2adb6486e2d55e4ac92dc75f}{Order\+Source\+::\+COMPOSITE\+\_\+\+ASK}, \doxylink{class_order_source_a423546ec13320cbc069f48f94ce689e1}{Order\+Source\+::\+REGIONAL\+\_\+\+BID}, and \doxylink{class_order_source_a39021c0afedcdade384daaadfa8f920b}{Order\+Source\+::\+REGIONAL\+\_\+\+ASK} are provided for convenience of a consolidated order book and are automatically generated based on the corresponding \doxylink{class_quote}{Quote} events. 
\item {\itshape Aggregate} sources \doxylink{class_order_source_a56c96834b2987e6ea3cb02bd1002394c}{Order\+Source\+::\+AGGREGATE\+\_\+\+BID} and \doxylink{class_order_source_a164e1f90d334b032429689cdbb563c22}{Order\+Source\+::\+AGGREGATE\+\_\+\+ASK} provide futures depth (aggregated by price level) and NASDAQ Level II (top of book for each market maker). These source cannot be directly published to via dx\+Feed API. 
\item {\itshape Publishable} sources \doxylink{class_order_source_aae5e89dc5fa5f9fc9cdbd9a70f9b7104}{Order\+Source\+::\+DEFAULT}, \doxylink{class_order_source_a082ff436a692c22800fb4cb54e9e7d58}{Order\+Source\+::\+NTV}, \doxylink{class_order_source_a91f76f9fbfccf2fbccecb9574287ecfb}{Order\+Source\+::\+ISE} etc support full range of dx\+Feed API features. 
\end{DoxyItemize}
