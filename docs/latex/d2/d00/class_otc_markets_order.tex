\doxysubsubsection{Detailed Description}
Represents an extension of \doxylink{class_order}{Order} for the symbols traded on the OTC Markets. 

It includes the OTC Markets specific quote data, such as \doxylink{class_otc_markets_order_a85155c31cda80ccd911f84389efcd666}{Quote Access Payment (QAP) value}, \doxylink{class_otc_markets_order_a1e69108a5349e0b819f4a5e0026d1e6e}{Quote Price Type}, \doxylink{class_otc_markets_order_ac37e7a0ebb4051f965125e608640165d}{Open/\+Closed status}, and \doxylink{class_otc_markets_order_abdc79206ae54aaa371a6ee46593cabc9}{Unsolicited quote indicator}. The original event (OTC \doxylink{class_quote}{Quote}) published by the exchange is presented in the form of two separate OTCMarkets\+Order events (bid and ask side). Note that the description of the fields (e.\+g. QAP) uses the original "{}quote"{} term. 

For more information about original fields, QAP, \doxylink{class_quote}{Quote} Flags and Extended \doxylink{class_quote}{Quote} Flags, see \href{https://downloads.dxfeed.com/specifications/OTC_Markets_Multicast_Data_Feeds.pdf}{\texttt{ OTC Markets Multicast Data Feed}}. 

For more information about display requirements, see \href{https://downloads.dxfeed.com/specifications/OTC_Markets_Data_Display_Requirements.pdf}{\texttt{ OTC Markets Display Requirements}}.

Like regular orders, OTC Markets order events arrive from multiple sources for the same market symbol and are distinguished by their \doxylink{class_order_base_a21bc36fb881577927015977bc5f71ec6}{index}. It is unique across all the sources of depth information for the symbol. The event with \doxylink{class_order_base_a1a128e82af72d3fcd44a7f3eb8bcfd44}{size} either {\ttfamily 0} or {\ttfamily NaN} is a signal to remove previously received order for the corresponding index. The method \doxylink{class_order_base_afb5bea9f67fcdd11410562ab462b901d}{has\+Size()} is a convenient method to test for size presence.

\doxysubsubsubsubsection*{\label{class_underlying_eventFlagsSection}%
\Hypertarget{class_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

OTC Markets order event sources provide a consistent view of the order book. Their updates may incorporate multiple changes to individual orders that have to be processed at the same time. The corresponding information is carried in \doxylink{class_order_base_af5e6a4420f059a8e46e664dafebb6ea4}{event\+Flags} property.

See the "{}\+Event Flags"{} section of \doxylink{class_order_base}{Order\+Base} class documentation for details.

\doxysubsubsubsubsection*{Publishing order books}

When publishing an order event with \doxylink{struct_d_x_publisher_ae6e8cb6b370ea60f1ad5cb80cea2df30}{DXPublisher\+::publish\+Events()} method, least significant 32 bits of order \doxylink{class_order_base_a21bc36fb881577927015977bc5f71ec6}{index} must be in a range of from 0 to {\ttfamily std\+::numeric\+\_\+limits\texorpdfstring{$<$}{<}std\+::int32\+\_\+t\texorpdfstring{$>$}{>}::max()} inclusive. Use \doxylink{class_order_base_a3451d2eb5655cd3c902c78e3368ecfe3}{set\+Source()} method after \doxylink{class_order_base_a1a7ce2fb19329a2f8450110cb58b9ac0}{set\+Index()} to properly include source identifier into the index.

A snapshot has to be published in the {\itshape descending} order of \doxylink{class_order_base_a21bc36fb881577927015977bc5f71ec6}{index}, starting with an event with the largest index and marking it with \doxylink{class_event_flag_ab1246c3963485123672b05d5b9dbdfec}{Event\+Flag\+::\+SNAPSHOT\+\_\+\+BEGIN} bit in \doxylink{class_order_base_af5e6a4420f059a8e46e664dafebb6ea4}{event\+Flags}, and finishing the snapshot with an event that has zero 32 least significant bits of index. \doxylink{class_event_flag_adc2ec8ed07a2a2efaf2389fe4f7459c6}{Event\+Flag\+::\+SNAPSHOT\+\_\+\+END} bit in \doxylink{class_order_base_af5e6a4420f059a8e46e664dafebb6ea4}{event\+Flags} is optional during publishing. It will be properly set on receiving end anyway.

\doxysubsubsubsubsection*{Limitations}

This event type cannot be used with DXFeed\+::get\+Last\+Event() method.
