\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{classCandle}{Candle}} alignment attribute of \mbox{\hyperlink{structCandleSymbol}{Candle\+Symbol}} defines how candle are aligned with respect to time. 

\doxysubparagraph*{Implementation details}

This attribute is encoded in a symbol string with \mbox{\hyperlink{structMarketEventSymbols_a87a94614e292548132480d85a1dfbe9d}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}}, \mbox{\hyperlink{structMarketEventSymbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}}, and \mbox{\hyperlink{structMarketEventSymbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key}} methods. The key to use with these methods is available via \mbox{\hyperlink{structCandleAlignment_a8ce587e43a5f8496c1a8ea0bf24b9499}{Candle\+Alignment\+::\+ATTRIBUTE\+\_\+\+KEY}} constant. The value that this key shall be set to is equal to the corresponding \mbox{\hyperlink{structCandleAlignment_aaad3e93c3271d2ece752681fb490ed57}{Candle\+Alignment\+::to\+String()}}
