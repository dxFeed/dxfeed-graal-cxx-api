\doxysubsubsection{Detailed Description}
Theo price is a snapshot of the theoretical option price computation that is periodically performed by \href{http://www.devexperts.com/en/products/price.html}{\texttt{ dx\+Price}} model-\/free computation. 

It represents the most recent information that is available about the corresponding values at any given moment of time. The values include first and second order derivative of the price curve by price, so that the real-\/time theoretical option price can be estimated on real-\/time changes of the underlying price in the vicinity.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \doxylink{classdxfcpp_1_1_theo_price}{Theo\+Price} sources provide a consistent view of the set of known \doxylink{classdxfcpp_1_1_theo_price}{Theo\+Price}. The corresponding information is carried in \doxylink{classdxfcpp_1_1_theo_price_a0c336c08711ef6c5ae0758f5bd47a3b0}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for \doxylink{classdxfcpp_1_1_theo_price}{Theo\+Price}, thus \doxylink{classdxfcpp_1_1_theo_price_a41ac1ef1fc6b203ae1230c542f0eebd6}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{classdxfcpp_1_1_theo_price_ae0613b3cd9e24ed38687e6b99b4fdc14}{time}.

\doxyparagraph*{Publishing \doxylink{classdxfcpp_1_1_theo_price}{Theo\+Price}}

Publishing of \doxylink{classdxfcpp_1_1_theo_price}{Theo\+Price} events follows the general rules explained in \doxylink{structdxfcpp_1_1_time_series_event}{Time\+Series\+Event} class documentation.

\doxyparagraph*{Implementation details}

This event is implemented on top of QDS records {\ttfamily \doxylink{classdxfcpp_1_1_theo_price}{Theo\+Price}}.
