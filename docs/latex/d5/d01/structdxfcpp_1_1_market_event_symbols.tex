\doxysubsubsection{Detailed Description}
Helper class to compose and parse symbols for market events. 

\doxyparagraph*{Regional symbols}

Regional symbol subscription receives events only from a designated exchange, marketplace, or venue instead of receiving composite events from all venues (by default). Regional symbol is composed from a {\itshape base symbol}, ampersand character (\textquotesingle{}\&\textquotesingle{}), and an exchange code character. For example, 
\begin{DoxyItemize}
\item "{}\+SPY"{} is the symbol for composite events for SPDR S\&P 500 ETF from all exchanges, 
\item "{}\+SPY\&\+N"{} is the symbol for event for SPDR S\&P 500 ETF that originate only from NYSE marketplace. 
\end{DoxyItemize}

\doxyparagraph*{Symbol attributes}

Market event symbols can have a number of attributes attached to then in curly braces with {\ttfamily \texorpdfstring{$<$}{<}key\texorpdfstring{$>$}{>}=\texorpdfstring{$<$}{<}value\texorpdfstring{$>$}{>}} paris separated by commas. For example, 
\begin{DoxyItemize}
\item "{}\+SPY\{price=bid\}"{} is the market symbol "{}\+SPY"{} with an attribute key "{}price"{} set to value "{}bid"{}. 
\item "{}\+SPY(=5m,tho=true\}"{} is the market symbol "{}\+SPY"{} with two attributes. One has an empty key and value "{}5m"{}, while the other has key "{}tho"{} and value "{}true"{}. 
\end{DoxyItemize}The methods in this class always maintain attribute keys in alphabetic order.
