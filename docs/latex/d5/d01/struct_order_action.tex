\doxysubsubsection{Detailed Description}
Action enum for the Full \doxylink{class_order}{Order} Book (FOB) Orders. 

Action describes business meaning of the \doxylink{class_order}{Order} event\+: whether order was added or replaced, partially or fully executed, etc.
