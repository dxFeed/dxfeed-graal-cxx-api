\doxysubsubsection{Detailed Description}
\doxylink{class_candle}{Candle} alignment attribute of \doxylink{struct_candle_symbol}{Candle\+Symbol} defines how candle are aligned with respect to time. 

\doxysubsubsubsubsection*{Implementation details}

This attribute is encoded in a symbol string with \doxylink{struct_market_event_symbols_a8eea112429ea61730be80e4fac021095}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}, \doxylink{struct_market_event_symbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}, and \doxylink{struct_market_event_symbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key} methods. The key to use with these methods is available via \doxylink{struct_candle_alignment_a8ce587e43a5f8496c1a8ea0bf24b9499}{Candle\+Alignment\+::\+ATTRIBUTE\+\_\+\+KEY} constant. The value that this key shall be set to is equal to the corresponding \doxylink{struct_candle_alignment_aaad3e93c3271d2ece752681fb490ed57}{Candle\+Alignment\+::to\+String()}
