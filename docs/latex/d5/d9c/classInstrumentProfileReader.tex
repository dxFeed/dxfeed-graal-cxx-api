\doxysubsubsection{Detailed Description}
Reads instrument profiles from the stream using Instrument \mbox{\hyperlink{classProfile}{Profile}} Format (IPF). 

Please see {\bfseries{Instrument \mbox{\hyperlink{classProfile}{Profile}} Format}} documentation for complete description. This reader automatically uses data formats as specified in the stream.

This reader is intended for \char`\"{}one time only\char`\"{} usage\+: create new instances for new IPF reads. 

Use \mbox{\hyperlink{classInstrumentProfileConnection}{Instrument\+Profile\+Connection}} if support for streaming updates of instrument profiles is needed.

For backward compatibility reader can be configured with system property \char`\"{}-\/\+Dcom.\+dxfeed.\+ipf.\+complete\char`\"{} to control the strategy for missing \char`\"{}\#\#\+COMPLETE\char`\"{} tag when reading IPF, possible values are\+: 
\begin{DoxyItemize}
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{warn }

\end{DoxyCode}
 -\/ show warning in the log (default) 
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{error }

\end{DoxyCode}
 -\/ throw exception (future default) 
\item 
\begin{DoxyCode}{0}
\DoxyCodeLine{ignore }

\end{DoxyCode}
 -\/ do nothing (for backward compatibility) 
\end{DoxyItemize}
