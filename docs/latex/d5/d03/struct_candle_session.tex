\doxysubsubsection{Detailed Description}
\doxylink{struct_session}{Session} attribute of \doxylink{struct_candle_symbol}{Candle\+Symbol} defines trading that is used to build the candles. 

\doxysubsubsubsubsection*{Implementation details}

This attribute is encoded in a symbol string with \doxylink{struct_market_event_symbols_a8eea112429ea61730be80e4fac021095}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}, \doxylink{struct_market_event_symbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}, and \doxylink{struct_market_event_symbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key} methods.

\doxylink{struct_candle_session_a67d891c75a8f8571d1c63d09615c98c6}{Candle\+Session\+::\+ANY} session is a default. The key to use with these methods is available via \doxylink{struct_candle_session_a9711ee826ebf3527e69f5e01725ca5e4}{Candle\+Session\+::\+ATTRIBUTE\+\_\+\+KEY} constant. The value that this key shall be set to is equal to the corresponding \doxylink{struct_candle_session_a01d65e306c5af0cc2c8116a736f543c2}{Candle\+Session\+::to\+String()}
