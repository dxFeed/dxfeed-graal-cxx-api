\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{classUnderlying}{Underlying}} event is a snapshot of computed values that are available for an option underlying symbol based on the option prices on the market. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxysubparagraph*{\label{_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \mbox{\hyperlink{classUnderlying}{Underlying}} sources provide a consistent view of the set of known \mbox{\hyperlink{classUnderlying}{Underlying}} events. The corresponding information is carried in event\+Flags property. The logic behind this property is detailed in \mbox{\hyperlink{structIndexedEvent}{Indexed\+Event}} class documentation. Multiple event sources for the same symbol are not supported for \mbox{\hyperlink{classUnderlying}{Underlying}}, thus source property is always \mbox{\hyperlink{classIndexedEventSource_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their time.

\doxysubparagraph*{Publishing \mbox{\hyperlink{classUnderlying}{Underlying}}}

Publishing of \mbox{\hyperlink{classUnderlying}{Underlying}} events follows the general rules explained in \mbox{\hyperlink{structTimeSeriesEvent}{Time\+Series\+Event}} class documentation.

\doxysubparagraph*{Implementation details}

This event is implemented on top of QDS record {\ttfamily \mbox{\hyperlink{classUnderlying}{Underlying}}}.
