\doxysubsubsection{Detailed Description}
Represents an indexed collection of up-\/to-\/date information about some condition or state of an external entity that updates in real-\/time. 

For example, \doxylink{class_order}{Order} represents an order to buy or to sell some market instrument that is currently active on a market exchange and multiple orders are active for each symbol at any given moment in time. \doxylink{class_candle}{Candle} represent snapshots of aggregate information about trading over a specific time period and there are multiple periods available. The \doxylink{class_candle}{Candle} is also an example of \doxylink{struct_time_series_event}{Time\+Series\+Event} that is a more specific event type.

Index for each event is available via \doxylink{struct_indexed_event_a4310a26785d5142efba9d7c0094125bc}{index} property. Indexed events retain information about multiple events per symbol based on the event index and are conflated based on the event index. Last indexed event for each symbol and index is always delivered to event listeners on subscription, but intermediate (next-\/to-\/last) events for each symbol+index pair are not queued anywhere, they are simply discarded as stale events. More recent events represent an up-\/to-\/date information about some external entity.

Some indexed events, like \doxylink{class_order}{Order}, support multiple sources of information for the same symbol. The event source is available via \doxylink{struct_indexed_event_a164c9e1c30069b14fad183ee1ae5b361}{source} property. The value of \doxylink{struct_indexed_event_a164c9e1c30069b14fad183ee1ae5b361}{source} property is always \doxylink{class_indexed_event_source_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT} for time-\/series events and other singe-\/sourced events like \doxylink{class_series}{Series}, that do not support this feature.
