\doxysubsubsection{Detailed Description}
Spread order event is a snapshot for a full available market depth for all spreads on a given underlying symbol. 

The collection of spread order events of a symbol represents the most recent information that is available about spread orders on the market at any given moment of time.

Spread order is similar to a regular \doxylink{classdxfcpp_1_1_order}{Order}, but it has a \doxylink{classdxfcpp_1_1_spread_order_a519138942e57e77a82587b4a01244db0}{spread\+Symbol} property that contains the symbol of the actual spread that is being represented by spread order object. \doxylink{structdxfcpp_1_1_market_event_a62b3475e3dc13476c57553cc354a2915}{event\+Symbol} property contains the underlying symbol that was used in subscription.

Like regular orders, spread order events arrive from multiple sources for the same market symbol and are distinguished by their \doxylink{structdxfcpp_1_1_indexed_event_ad5a71500687f8cebb2a15964144638dc}{index}. It is unique across all the sources of depth information for the symbol. The event with \doxylink{classdxfcpp_1_1_order_base_a3772a63a9d2a79b8cb9a687a2b2c0a7a}{size} either {\ttfamily 0} or {\ttfamily NaN} is a signal to remove previously received order for the corresponding index. The method \doxylink{classdxfcpp_1_1_order_base_aa7c935513d5717625af3563761703350}{has\+Size} is a convenient method to test for size presence.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some spread order event sources provide a consistent view of the price-\/level or detailed order book. Their updates may incorporate multiple changes to price levels or to individual orders that have to be processed at the same time. The corresponding information is carried in \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{event\+Flags} property.

See {\ttfamily Event Flags} section of \doxylink{classdxfcpp_1_1_order_base}{Order\+Base} class documentation for details.

Indexed\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of events. It relies on the code of Abstract\+Indexed\+Event\+Model to handle this logic. Use the source code of Abstract\+Indexed\+Event\+Model for clarification on transactions and snapshot logic.

\doxyparagraph*{Publishing order books}

When publishing an order event with \doxylink{structdxfcpp_1_1_d_x_publisher_a9ec3ba1e21e011c21c5d9f30c21305f1}{DXPublisher\+::publish\+Events()} method, least significant 32 bits of order \doxylink{structdxfcpp_1_1_indexed_event_ad5a71500687f8cebb2a15964144638dc}{index} must be in a range of from 0 to {\ttfamily std\+::numeric\+\_\+limits\texorpdfstring{$<$}{<}std\+::int32\+\_\+t\texorpdfstring{$>$}{>}::\doxylink{structdxfcpp_1_1_shared_entity_aa117214d0b4e4fc6979400eff315a8d3}{max()}} inclusive. Use \doxylink{classdxfcpp_1_1_order_base_a1813c11582e335c3654c028ad80b7a0d}{set\+Source} method after \doxylink{structdxfcpp_1_1_indexed_event_a5467b31a73a69502b6590656980b108d}{set\+Index} to properly include source identifier into the index.

A snapshot has to be published in the {\itshape descending} order of \doxylink{structdxfcpp_1_1_indexed_event_ad5a71500687f8cebb2a15964144638dc}{index}, starting with an event with the largest index and marking it with \doxylink{structdxfcpp_1_1_indexed_event_aacdbdadf8af938b9edbe9a59830aa0b5}{Spread\+Order\+::\+SNAPSHOT\+\_\+\+BEGIN} bit in \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{event\+Flags}, and finishing the snapshot with an event that has zero 32 least significant bits of index. \doxylink{structdxfcpp_1_1_indexed_event_a8140c207eb5efb3f656842509700366f}{Spread\+Order\+::\+SNAPSHOT\+\_\+\+END} bit in \doxylink{structdxfcpp_1_1_indexed_event_a45ece86bfc828bf8a40ef1def0ec625e}{event\+Flags} is optional during publishing. It will be properly set on receiving end anyway.

\doxyparagraph*{Limitations}

This event type cannot be used with \doxylink{structdxfcpp_1_1_shared_entity_aa117214d0b4e4fc6979400eff315a8d3}{DXFeed\+::get\+Last\+Event()} method.
