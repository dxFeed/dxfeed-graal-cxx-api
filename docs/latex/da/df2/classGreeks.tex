\doxysubsubsection{Detailed Description}
\mbox{\hyperlink{classGreeks}{Greeks}} event is a snapshot of the option price, Black-\/\+Scholes volatility and greeks. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxysubparagraph*{\label{_eventFlagsSection}%
Event flags, transactions and snapshots}

Some greeks sources provide a consistent view of the set of known greeks. The corresponding information is carried in event\+Flags property. The logic behind this property is detailed in \mbox{\hyperlink{structIndexedEvent}{Indexed\+Event}} class documentation. Multiple event sources for the same symbol are not supported for greeks, thus source property is always \mbox{\hyperlink{classIndexedEventSource_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their time. It relies on the code of Abstract\+Indexed\+Event\+Model to handle this logic. Use the source code of Abstract\+Indexed\+Event\+Model for clarification on transactions and snapshot logic.

\doxysubparagraph*{Publishing \mbox{\hyperlink{classGreeks}{Greeks}}}

Publishing of greeks events follows the general rules explained in \mbox{\hyperlink{structTimeSeriesEvent}{Time\+Series\+Event}} class documentation.
