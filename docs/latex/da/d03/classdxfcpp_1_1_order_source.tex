\doxysubsubsection{Detailed Description}
Identifies source of \doxylink{classdxfcpp_1_1_order}{Order}, \doxylink{classdxfcpp_1_1_analytic_order}{Analytic\+Order}, Otc\+Markets\+Order and \doxylink{classdxfcpp_1_1_spread_order}{Spread\+Order} events. 

There are the following kinds of order sources\+: 
\begin{DoxyItemize}
\item {\itshape Synthetic} sources \doxylink{classdxfcpp_1_1_order_source_a1aac74d06a7073aebabbc94c7e1a694c}{Order\+Source\+::\+COMPOSITE\+\_\+\+BID}, \doxylink{classdxfcpp_1_1_order_source_a0dc70f8cb493322db0881baa63c2883f}{Order\+Source\+::\+COMPOSITE\+\_\+\+ASK}, \doxylink{classdxfcpp_1_1_order_source_a22d5b9593951fdad1e6cb57292c01e9e}{Order\+Source\+::\+REGIONAL\+\_\+\+BID}, and \doxylink{classdxfcpp_1_1_order_source_a909f1f24bdcf3ded814e363722b83278}{Order\+Source\+::\+REGIONAL\+\_\+\+ASK} are provided for convenience of a consolidated order book and are automatically generated based on the corresponding \doxylink{classdxfcpp_1_1_quote}{Quote} events. 
\item {\itshape Aggregate} sources \doxylink{classdxfcpp_1_1_order_source_a9290fe6ec805b6d7f1e654f70f2f4a9c}{Order\+Source\+::\+AGGREGATE\+\_\+\+BID} and \doxylink{classdxfcpp_1_1_order_source_a04c738279aeef41e9c19af191320559c}{Order\+Source\+::\+AGGREGATE\+\_\+\+ASK} provide futures depth (aggregated by price level) and NASDAQ Level II (top of book for each market maker). These source cannot be directly published to via dx\+Feed API. 
\item {\itshape Publishable} sources \doxylink{classdxfcpp_1_1_order_source_afd195a0f6279d0af47983ef32438f7bc}{Order\+Source\+::\+DEFAULT}, \doxylink{classdxfcpp_1_1_order_source_af2939f2b53dce77501e75af134935492}{Order\+Source\+::\+NTV}, \doxylink{classdxfcpp_1_1_order_source_a5830beb91b5bf0307547d5dbe45ca7be}{Order\+Source\+::\+ISE} etc support full range of dx\+Feed API features. 
\end{DoxyItemize}
