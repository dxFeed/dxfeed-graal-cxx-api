\doxysubsubsection{Detailed Description}
Represents an extension of \doxylink{class_order}{Order} introducing analytic information, e.\+g. 

adding to this order iceberg related information (\doxylink{class_analytic_order_a57ae9239ea997a589845f693f69bc17c}{iceberg\+Peak\+Size}, \doxylink{class_analytic_order_a21de1b88caf4eb9730d225909437a64e}{iceberg\+Hidden\+Size}, \doxylink{class_analytic_order_a9ec4f7b3e2ee5d252496cfceae8e9734}{iceberg\+Executed\+Size}). The collection of analytic order events of a symbol represents the most recent analytic information that is available about orders on the market at any given moment of time.

Analytic order is similar to a regular \doxylink{class_order}{Order}. In addition this event has few additional properties\+: 
\begin{DoxyItemize}
\item \doxylink{class_analytic_order_a57ae9239ea997a589845f693f69bc17c}{iceberg\+Peak\+Size} -\/ the size of the peak, i.\+e. the visible part of the iceberg, that is being continually refilled until the order is fully traded or cancelled; 
\item \doxylink{class_analytic_order_a21de1b88caf4eb9730d225909437a64e}{iceberg\+Hidden\+Size} -\/ the prediction of current hidden size of the iceberg, as inferred by the model; 
\item \doxylink{class_analytic_order_a9ec4f7b3e2ee5d252496cfceae8e9734}{iceberg\+Executed\+Size} -\/ the executed size of the iceberg order. For \doxylink{struct_iceberg_type_a0e29a6c8573c0d6b6cf33b17dbc70902}{Iceberg\+Type\+::\+SYNTHETIC} type represents total executed size of all orders composing current iceberg; 
\item \doxylink{class_analytic_order_a3ce248e64ff53c0dbacc1cb8927bfc21}{Analytic\+Order\+::get\+Iceberg\+Type()} -\/ type of the iceberg, either native (exchange-\/managed) or synthetic (managed outside of the exchange). 
\end{DoxyItemize}

\doxysubsubsubsubsection*{Implementation details}

This event is implemented on top of QDS records {\ttfamily \doxylink{class_analytic_order}{Analytic\+Order}\#\texorpdfstring{$<$}{<}source-\/id\texorpdfstring{$>$}{>}}, where {\ttfamily \texorpdfstring{$<$}{<}source-\/id\texorpdfstring{$>$}{>}} is up to 4 ASCII characters with a mnemonic for the source like "{}\+GLBX"{}.
