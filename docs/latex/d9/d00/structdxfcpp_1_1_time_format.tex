{\footnotesize\ttfamily std\+::int64\+\_\+t dxfcpp\+::\+Time\+Format\+::parse (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{value }\end{DoxyParamCaption}) const\hspace{0.3cm}{\ttfamily [noexcept]}}



Reads Date from String and returns timestamp. 

This method is designed to understand \href{http://en.wikipedia.org/wiki/ISO_8601}{\texttt{ ISO 8601}} formatted date and time. It accepts the following formats\+:


\begin{DoxyItemize}
\item {\bfseries{{\ttfamily 0}}} is parsed as zero time.  
\item {\bfseries{{\ttfamily \textquotesingle{}-\/\textquotesingle{}\texorpdfstring{$<$}{<}time-\/period\texorpdfstring{$>$}{>}}}} it is parsed as \texorpdfstring{$<$}{<}current time\texorpdfstring{$>$}{>} -\/ \texorpdfstring{$<$}{<}time-\/period\texorpdfstring{$>$}{>}.  
\item {\bfseries{{\ttfamily \texorpdfstring{$<$}{<}long-\/value-\/in-\/milliseconds\texorpdfstring{$>$}{>}}}} It should be positive and have at least 9 digits (otherwise it could not be distinguished from date in format {\ttfamily \textquotesingle{}yyyymmdd\textquotesingle{}}). Each date since 1970-\/01-\/03 can be represented in this form.  
\item {\bfseries{{\ttfamily \texorpdfstring{$<$}{<}date\texorpdfstring{$>$}{>}\mbox{[}(\textquotesingle{}T\textquotesingle{}\texorpdfstring{$\vert$}{|}\textquotesingle{}t\textquotesingle{}\texorpdfstring{$\vert$}{|}\textquotesingle{}-\/\textquotesingle{}\texorpdfstring{$\vert$}{|}\textquotesingle{} \textquotesingle{})\texorpdfstring{$<$}{<}time\texorpdfstring{$>$}{>}\mbox{]}\mbox{[}\texorpdfstring{$<$}{<}timezone\texorpdfstring{$>$}{>}\mbox{]}}}} If time is missing it is supposed to be {\ttfamily \textquotesingle{}00\+:00\+:00\textquotesingle{}}.  
\item {\bfseries{{\ttfamily \mbox{[}\textquotesingle{}T\textquotesingle{}\texorpdfstring{$\vert$}{|}\textquotesingle{}t\textquotesingle{}\mbox{]}\texorpdfstring{$<$}{<}time\texorpdfstring{$>$}{>}\mbox{[}\texorpdfstring{$<$}{<}timezone\texorpdfstring{$>$}{>}\mbox{]}}}} In this case current date is used.  
\end{DoxyItemize}Here
