\doxysubsubsection{Detailed Description}
Period attribute of \doxylink{struct_candle_symbol}{Candle\+Symbol} defines aggregation period of the candles. 

Aggregation period is defined as pair of a \doxylink{struct_candle_period_a1e2e33ae5421ab04b022a50db2a4ad0a}{value} and \doxylink{struct_candle_period_a89aef310d793d619458f1afa53166402}{type}.

\doxysubsubsubsubsection*{Implementation details}

This attribute is encoded in a symbol string with \doxylink{struct_market_event_symbols_a8eea112429ea61730be80e4fac021095}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}, \doxylink{struct_market_event_symbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}, and \doxylink{struct_market_event_symbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key} methods. The key to use with these methods is available via \doxylink{struct_candle_period_a2875205da52d8568bdbe57b6a59cf154}{Candle\+Period\+::\+ATTRIBUTE\+\_\+\+KEY} constant. The value that this key shall be set to is equal to the corresponding \doxylink{struct_candle_period_a9006fadf4e732459b6fcec7fe056fb4c}{Candle\+Period\+::to\+String()} "{}\+Candle\+Period.\+to\+String()"{}
