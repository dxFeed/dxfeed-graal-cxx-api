{\footnotesize\ttfamily std\+::int64\+\_\+t Time\+Format\+::parse (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{value }\end{DoxyParamCaption}) const}



Reads Date from String and returns timestamp. 

This method is designed to understand \href{http://en.wikipedia.org/wiki/ISO_8601}{\texttt{ ISO 8601}} formatted date and time. It accepts the following formats\+:


\begin{DoxyItemize}
\item {\bfseries{{\ttfamily 0}}} is parsed as zero time.  
\item {\bfseries{{\ttfamily \textquotesingle{}-\/\textquotesingle{}$<$time-\/period$>$}}} it is parsed as $<$current time$>$ -\/ $<$time-\/period$>$.  
\item {\bfseries{{\ttfamily $<$long-\/value-\/in-\/milliseconds$>$}}} It should be positive and have at least 9 digits (otherwise it could not be distinguished from date in format {\ttfamily \textquotesingle{}yyyymmdd\textquotesingle{}}). Each date since 1970-\/01-\/03 can be represented in this form.  
\item {\bfseries{{\ttfamily $<$date$>$\mbox{[}(\textquotesingle{}T\textquotesingle{}$\vert$\textquotesingle{}t\textquotesingle{}$\vert$\textquotesingle{}-\/\textquotesingle{}$\vert$\textquotesingle{} \textquotesingle{})$<$time$>$\mbox{]}\mbox{[}$<$timezone$>$\mbox{]}}}} If time is missing it is supposed to be {\ttfamily \textquotesingle{}00\+:00\+:00\textquotesingle{}}.  
\item {\bfseries{{\ttfamily \mbox{[}\textquotesingle{}T\textquotesingle{}$\vert$\textquotesingle{}t\textquotesingle{}\mbox{]}$<$time$>$\mbox{[}$<$timezone$>$\mbox{]}}}} In this case current date is used.  
\end{DoxyItemize}Here 
\begin{DoxyItemize}
\item {\bfseries{{\ttfamily $<$date$>$}}} is one of 
\begin{DoxyItemize}
\item {\bfseries{{\ttfamily yyyy-\/\+MM-\/dd}}} 
\item {\bfseries{{\ttfamily yyyy\+MMdd}}} 
\end{DoxyItemize}
\item {\bfseries{{\ttfamily $<$time$>$}}} is one of 
\begin{DoxyItemize}
\item {\bfseries{{\ttfamily HH\+:mm\+:ss\mbox{[}.sss\mbox{]}}}} 
\item {\bfseries{{\ttfamily HHmmss\mbox{[}.sss\mbox{]}}}} 
\item {\bfseries{{\ttfamily HH\+:mm}}} 
\item {\bfseries{{\ttfamily HHmm}}} 
\item {\bfseries{{\ttfamily HH}}} 
\end{DoxyItemize}
\item {\bfseries{{\ttfamily $<$timezone$>$}}} is one of 
\begin{DoxyItemize}
\item {\bfseries{{\ttfamily \mbox{[}+-\/\mbox{]}HH\+:mm}}} 
\item {\bfseries{{\ttfamily \mbox{[}+-\/\mbox{]}HHmm}}} 
\item {\bfseries{{\ttfamily \mbox{[}+-\/\mbox{]}HH}}} 
\item {\bfseries{{\ttfamily Z}}} for UTC. 
\item or any timezone that can be parsed by Java Simple\+Date\+Format. 
\end{DoxyItemize}
\end{DoxyItemize}
