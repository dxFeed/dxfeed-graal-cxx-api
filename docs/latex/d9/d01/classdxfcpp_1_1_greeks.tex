\doxysubsubsection{Detailed Description}
\doxylink{classdxfcpp_1_1_greeks}{Greeks} event is a snapshot of the option price, Black-\/\+Scholes volatility and greeks. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxyparagraph*{\label{classdxfcpp_1_1_underlying_eventFlagsSection}%
\Hypertarget{classdxfcpp_1_1_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some greeks sources provide a consistent view of the set of known greeks. The corresponding information is carried in \doxylink{classdxfcpp_1_1_greeks_aec185ec2c069dc9ea59266dcbca519f2}{event\+Flags} property. The logic behind this property is detailed in \doxylink{structdxfcpp_1_1_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for greeks, thus \doxylink{classdxfcpp_1_1_greeks_ac02eb424b84ac82345cf01bcd443c9cb}{source} property is always \doxylink{classdxfcpp_1_1_indexed_event_source_ab7bc1a5037c8e18b2adeb45237cea5eb}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{classdxfcpp_1_1_greeks_a65ed5282ee947f0d50f67acc14c072f2}{time}. It relies on the code of Abstract\+Indexed\+Event\+Model to handle this logic. Use the source code of Abstract\+Indexed\+Event\+Model for clarification on transactions and snapshot logic.

\doxyparagraph*{Publishing \doxylink{classdxfcpp_1_1_greeks}{Greeks}}

Publishing of greeks events follows the general rules explained in \doxylink{structdxfcpp_1_1_time_series_event}{Time\+Series\+Event} class documentation.
