\doxysubsubsection{Detailed Description}
Period attribute of \mbox{\hyperlink{structCandleSymbol}{Candle\+Symbol}} defines aggregation period of the candles. 

Aggregation period is defined as pair of a \mbox{\hyperlink{structCandlePeriod_a1e2e33ae5421ab04b022a50db2a4ad0a}{value}} and \mbox{\hyperlink{structCandlePeriod_ad3a01c476df66e8f88e0a92752c180a0}{type}}.

\doxysubparagraph*{Implementation details}

This attribute is encoded in a symbol string with \mbox{\hyperlink{structMarketEventSymbols_a87a94614e292548132480d85a1dfbe9d}{Market\+Event\+Symbols.get\+Attribute\+String\+By\+Key}}, \mbox{\hyperlink{structMarketEventSymbols_ab4184832b9b58374117d61b1f8c94617}{change\+Attribute\+String\+By\+Key}}, and \mbox{\hyperlink{structMarketEventSymbols_ab0369d51b1181a4fb7ccb674923ddb9d}{remove\+Attribute\+String\+By\+Key}} methods. The key to use with these methods is available via \mbox{\hyperlink{structCandlePeriod_a2875205da52d8568bdbe57b6a59cf154}{Candle\+Period\+::\+ATTRIBUTE\+\_\+\+KEY}} constant. The value that this key shall be set to is equal to the corresponding \mbox{\hyperlink{structCandlePeriod_af79df084a414944a3de776502dcff3b5}{Candle\+Period\+::to\+String()}} \char`\"{}\+Candle\+Period.\+to\+String()\char`\"{}
