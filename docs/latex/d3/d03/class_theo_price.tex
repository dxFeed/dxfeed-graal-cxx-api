\doxysubsubsection{Detailed Description}
Theo price is a snapshot of the theoretical option price computation that is periodically performed by \href{http://www.devexperts.com/en/products/price.html}{\texttt{ dx\+Price}} model-\/free computation. 

It represents the most recent information that is available about the corresponding values at any given moment of time. The values include first and second order derivative of the price curve by price, so that the real-\/time theoretical option price can be estimated on real-\/time changes of the underlying price in the vicinity.

\doxysubsubsubsubsection*{\label{class_underlying_eventFlagsSection}%
\Hypertarget{class_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \doxylink{class_theo_price}{Theo\+Price} sources provide a consistent view of the set of known \doxylink{class_theo_price}{Theo\+Price}. The corresponding information is carried in \doxylink{class_theo_price_a395c746a2bf5523b2d66c72a1fc2feab}{event\+Flags} property. The logic behind this property is detailed in \doxylink{struct_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for \doxylink{class_theo_price}{Theo\+Price}, thus \doxylink{class_theo_price_a5db2cf3d8c3b6bd7298871294cd44e68}{source} property is always \doxylink{class_indexed_event_source_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{class_theo_price_a558ade41de4e5030d761a667fe495def}{time}.

Publishing of \doxylink{class_theo_price}{Theo\+Price} events follows the general rules explained in \doxylink{struct_time_series_event}{Time\+Series\+Event} class documentation.
