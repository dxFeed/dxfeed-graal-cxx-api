\doxysubsubsection{Detailed Description}
\doxylink{class_underlying}{Underlying} event is a snapshot of computed values that are available for an option underlying symbol based on the option prices on the market. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxysubsubsubsubsection*{\label{class_underlying_eventFlagsSection}%
\Hypertarget{class_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some \doxylink{class_underlying}{Underlying} sources provide a consistent view of the set of known \doxylink{class_underlying}{Underlying} events. The corresponding information is carried in \doxylink{class_underlying_a79bc065e8e06ea8947b7c83558d87bc2}{event\+Flags} property. The logic behind this property is detailed in \doxylink{struct_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for \doxylink{class_underlying}{Underlying}, thus \doxylink{class_underlying_a38e72fa1568091f8154b044e599b709e}{source} property is always \doxylink{class_indexed_event_source_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{class_underlying_a6f0e627f1f3ad7b7c5a47ca9e5adf36e}{time}.

\doxysubsubsubsubsection*{Publishing \doxylink{class_underlying}{Underlying}}

Publishing of \doxylink{class_underlying}{Underlying} events follows the general rules explained in \doxylink{struct_time_series_event}{Time\+Series\+Event} class documentation.

\doxysubsubsubsubsection*{Implementation details}

This event is implemented on top of QDS record {\ttfamily \doxylink{class_underlying}{Underlying}}.
