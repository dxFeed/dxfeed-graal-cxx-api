\doxysubsubsection{Detailed Description}
Action enum for the Full \mbox{\hyperlink{classOrder}{Order}} Book (FOB) Orders. 

Action describes business meaning of the \mbox{\hyperlink{classOrder}{Order}} event\+: whether order was added or replaced, partially or fully executed, etc.
