Adds subscription change listener. 

This method does nothing if subscription is closed. Otherwise, it installs the corresponding listener and immediately invokes Observable\+Subscription\+Change\+Listener\+::symbols\+Added() on the given listener while holding the lock for this subscription. This way the given listener synchronously receives existing subscription state and is synchronously notified on all changes in subscription afterwards.

Removes subscription change listener by id. 

This method does nothing if the listener with the given id was not installed or was already removed as subscription change listener for this subscription. Otherwise it removes the corresponding listener and immediately invokes Observable\+Subscription\+Change\+Listener\+::subscription\+Closed() on the given listener while holding the lock for this subscription.
