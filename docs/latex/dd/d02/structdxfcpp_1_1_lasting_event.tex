\doxysubsubsection{Detailed Description}
Represents up-\/to-\/date information about some condition or state of an external entity that updates in real-\/time. 

For example, a \doxylink{classdxfcpp_1_1_quote}{Quote} is an up-\/to-\/date information about best bid and best offer for a specific symbol.

Lasting events are conflated for each symbol. Last event for each symbol is always delivered to event listeners on subscription, but intermediate (next-\/to-\/last) events are not queued anywhere, they are simply discarded as stale events. More recent events represent an up-\/to-\/date information about some external entity.

Lasting events can be used with \doxylink{structdxfcpp_1_1_shared_entity_aa117214d0b4e4fc6979400eff315a8d3}{DXFeed\+::get\+Last\+Event()} and \doxylink{structdxfcpp_1_1_shared_entity_aa117214d0b4e4fc6979400eff315a8d3}{DXFeed\+::get\+Last\+Events} methods to retrieve last events for each symbol.

Note, that subscription to all lasting events of a specific type via \doxylink{structdxfcpp_1_1_wildcard_symbol_a7c4084f144af571c8ac73b68c49dbc5c}{Wildcard\+Symbol\+::\+ALL} symbol object does not benefit from the above advantages of lasting events.
