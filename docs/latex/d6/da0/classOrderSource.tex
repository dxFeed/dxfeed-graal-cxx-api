\doxysubsubsection{Detailed Description}
Identifies source of \mbox{\hyperlink{classOrder}{Order}}, \mbox{\hyperlink{classAnalyticOrder}{Analytic\+Order}}, \mbox{\hyperlink{classOtcMarketsOrder}{Otc\+Markets\+Order}} and \mbox{\hyperlink{classSpreadOrder}{Spread\+Order}} events. 

There are the following kinds of order sources\+: 
\begin{DoxyItemize}
\item {\itshape Synthetic} sources \mbox{\hyperlink{classOrderSource_af9c7de7588af5d798c2857af264c4e47}{Order\+Source\+::\+COMPOSITE\+\_\+\+BID}}, \mbox{\hyperlink{classOrderSource_a7197bc1f2adb6486e2d55e4ac92dc75f}{Order\+Source\+::\+COMPOSITE\+\_\+\+ASK}}, \mbox{\hyperlink{classOrderSource_a423546ec13320cbc069f48f94ce689e1}{Order\+Source\+::\+REGIONAL\+\_\+\+BID}}, and \mbox{\hyperlink{classOrderSource_a39021c0afedcdade384daaadfa8f920b}{Order\+Source\+::\+REGIONAL\+\_\+\+ASK}} are provided for convenience of a consolidated order book and are automatically generated based on the corresponding \mbox{\hyperlink{classQuote}{Quote}} events. 
\item {\itshape Aggregate} sources \mbox{\hyperlink{classOrderSource_a56c96834b2987e6ea3cb02bd1002394c}{Order\+Source\+::\+AGGREGATE\+\_\+\+BID}} and \mbox{\hyperlink{classOrderSource_a164e1f90d334b032429689cdbb563c22}{Order\+Source\+::\+AGGREGATE\+\_\+\+ASK}} provide futures depth (aggregated by price level) and NASDAQ Level II (top of book for each market maker). These source cannot be directly published to via dx\+Feed API. 
\item {\itshape Publishable} sources \mbox{\hyperlink{classOrderSource_aae5e89dc5fa5f9fc9cdbd9a70f9b7104}{Order\+Source\+::\+DEFAULT}}, \mbox{\hyperlink{classOrderSource_a082ff436a692c22800fb4cb54e9e7d58}{Order\+Source\+::\+NTV}}, \mbox{\hyperlink{classOrderSource_a91f76f9fbfccf2fbccecb9574287ecfb}{Order\+Source\+::\+ISE}} etc support full range of dx\+Feed API features. 
\end{DoxyItemize}
