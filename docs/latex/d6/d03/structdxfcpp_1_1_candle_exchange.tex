\doxysubsubsection{Detailed Description}
Exchange attribute of \doxylink{structdxfcpp_1_1_candle_symbol}{Candle\+Symbol} defines exchange identifier where data is taken from to build the candles. 

\doxyparagraph*{Implementation details}
