The simplest address strings have the following format\+:
\begin{DoxyItemize}
\item {\ttfamily host\+:port} to establish a TCP/\+IP connection.
\item {\ttfamily \+:port} to listen for a TCP/\+IP connection with a plain socket connector (good for up to a few hundred of connections).
