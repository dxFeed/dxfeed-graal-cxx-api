\doxysubsubsection{Detailed Description}
\doxylink{class_greeks}{Greeks} event is a snapshot of the option price, Black-\/\+Scholes volatility and greeks. 

It represents the most recent information that is available about the corresponding values on the market at any given moment of time.

\doxysubsubsubsubsection*{\label{class_underlying_eventFlagsSection}%
\Hypertarget{class_underlying_eventFlagsSection}%
Event flags, transactions and snapshots}

Some greeks sources provide a consistent view of the set of known greeks. The corresponding information is carried in \doxylink{class_greeks_a8e9fbcf81dd2fb99b624b0693bf38b58}{event\+Flags} property. The logic behind this property is detailed in \doxylink{struct_indexed_event}{Indexed\+Event} class documentation. Multiple event sources for the same symbol are not supported for greeks, thus \doxylink{class_greeks_a015ac9ffbd575c968fef29881519242a}{source} property is always \doxylink{class_indexed_event_source_ab359b82d74ee151707106b66eec0a4e3}{DEFAULT}.

Time\+Series\+Event\+Model class handles all the snapshot and transaction logic and conveniently represents a list current of time-\/series events order by their \doxylink{class_greeks_abfa6973297c02cfc7d4f160d8cc919ed}{time}. It relies on the code of Abstract\+Indexed\+Event\+Model to handle this logic. Use the source code of Abstract\+Indexed\+Event\+Model for clarification on transactions and snapshot logic.

\doxysubsubsubsubsection*{Publishing \doxylink{class_greeks}{Greeks}}

Publishing of greeks events follows the general rules explained in \doxylink{struct_time_series_event}{Time\+Series\+Event} class documentation.
